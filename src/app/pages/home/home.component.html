The Sapphire Testnet

Ethereum Serenity Phase 0 Test Network

Become a Validator

The Sapphire Testnet is a public network created by Prysmatic Labs. It implements the Ethereum 2.0 Phase 0 protocol for a proof-of-stake blockchain enabling anyone holding Goerli test ETH to join and participate.

This is a highly experimental test network with no security guarantees. The network will have scheduled restarts for improvements and maintenance as we help make eth2 a reality.

Frequently Asked Questions

What is Prysm?

Built in Go and fully-open source, the network runs on an eth2 node implementation called Prysm. To participate in the network, anyone can send {{DEPOSIT_AMOUNT}} ETH from the Goerli ETH1.0 testnet into a validator deposit contract, which will queue in the user as a validator in the system. Validators participate in proposing and voting on blocks in the protocol, and the network applies rewards/penalties based on their behavior.

Why should I care?

Ethereum 2.0 is a massive, multi-year effort to create a true world computer. In other words, its goal is to create a truly decentralized, permissionless platform for programmable money through smart contracts. Phase 0, which the Prysm testnet implements, is a great step forward in making the years of research behind the future of Ethereum a tangible reality. This gives us a real, public network we can push to the limit and learn about ways to improve our approach before we consider a multi-client, mainnet release of the network.

How can I help?

We have a ton of work to do and this is only the beginning. Joining our discord is a great way to get involved, just introduce yourself and let us know what you want to contribute to. Also, check out our open issues on Github here.
